# Password-Generator
A simple password generator written in python that helps to generate a strong and unique password for a user.


# Features 
- Generates passwords of 12 length
-  Uses letter, numbers, and symbols

# Requirement 
- Python 3.x

# How to run 
1. Clone the repository:
https://github.com/Joyce0027/Password-Generator.git
2. cd Password-Generator
3. Run the script:
  python password_generator.py
